# `slides.nvim`

Slides, but in Neovim.

# Usage

Run `:StartSlides` in a markdown file.

Use `n` and `p` to navigate slides, and `q` to quit.

# Features

You can also use `X` to run code blocks

```lua
print("hi", 42)
```

# Even other languages

```js
console.log({hi: 42})
```
